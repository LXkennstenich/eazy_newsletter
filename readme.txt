Plugin for sending newsletters 

== Description ==

You can define your own custom html wrappers for the newsletter in the settings section. 
Just remember to close the elements containing the content of the newsletter in the footer html box because each part is appended one by one. 


There is also a shortcode available for the frontend form:

[eazy_newsletter]


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/eazy_newsletter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings->eazy_newsletter to configure the plugin
4. Change the name and the e-mail address for the "from" headers. the default settings will get detected as spam.

Define the settings like you want to.
